About Me 私について

戸高光太朗 Todaka Kotaro

1995年生まれ、京都府出身。ストーリー性のある見せ方をするWebデザインに興味を持ち、2022年3月よりデジタルハリウッドSTUDIO by LIGに入学。映像授業と課題を通してWebデザインとコーディングについて学ぶ。

Skill できること

  • Webデザイン Web Design

    ご依頼者さまの目的に沿った情報設計と、サイト利用者の使いやすさを重視したデザインを作ります。お客さまの商品・サービスのコンセプトを理解し、その雰囲気がしっかりと伝わる制作を心がけています。デザインを通して商品の魅力を引き出すことを目指しています。

    Tools
    Illustrator
    Photoshop
    XD

  • コーディング Coding

    HTML/CSSを使い、デザインを忠実に再現します。読み上げ機能やキーボード操作にも対応する設計を意識し、アクセシビリティに配慮しています。CSSアニメーションとJavaScriptを組み合わせたアニメーションにも力を入れており、デザインにプラスアルファの価値を付与します。

    Tools
    Visual Studio Code
    Dreamweaver
    Coding Language
    HTML
    CSS
    jQuery
    JavaScript

  • イラスト Illust

    写真からイラストをつくれます。絵柄の参考を提供していただくことで、その雰囲気に写真を変えることが得意です。簡単なアニメーション制作もでき、線画のみ、塗りだけのものなど、描き方が与える影響を考慮し、デザインのワンポイントになるようなイラスト制作を意識しています。

    Tools
    Illustrator
    Clip Studio Paint
